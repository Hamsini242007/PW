# Carbon Footprint Awareness Platform

## Problem Statement

Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.

---

## Project Overview

The Carbon Footprint Awareness Platform is an interactive web application that helps users understand the environmental impact of their daily activities through awareness content, activity tracking, personalized eco-scoring, and visual analytics.

The platform encourages sustainable habits by allowing users to record daily activities, monitor progress, and receive meaningful feedback on their environmental impact.
